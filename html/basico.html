Las HTML son las siglas de HyperText Markup Language.

Historia de HTML

Tim Berners-Lee definió el lenguaje HTML mientras trabajaba en el CERN en 1989.

Los inicios del lenguaje HTML se remontan a la propuesta de Berners-Lee de un sistema de gestión de información para el CERN.

Versiones de HTML

Tim Berners-Lee definió la primera versión de HTML en 1991.
